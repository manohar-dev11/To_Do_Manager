# 🚀 Smart Task Manager

A modern **full-stack productivity dashboard** for managing tasks with priorities, deadlines, tags, and analytics.

This project is being developed step-by-step as a **learning + portfolio project** while exploring frontend and backend development.

---

## ✨ Features (Current Progress)

* Add tasks
* Delete tasks
* Mark tasks as completed
* Dynamic task rendering
* Task data model using JavaScript objects

---

## 🔜 Upcoming Features

* Task priority sorting
* Search tasks
* Tag filtering
* Productivity analytics
* Backend API with Flask
* Persistent database storage
